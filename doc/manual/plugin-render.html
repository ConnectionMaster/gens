Gens/GS - Plugins - Render Plugins

Index → Plugins → Render Plugins

Gens/GS includes a wide assortment of rendering plugins. Two plugins, "Normal" and "Double", are built into the emulator, while the rest are external MDP plugins.

Normal

The "Normal" renderer is a simple 1x rendering plugin that blits the original video image as-is without any modifications.

Double

The "Double" renderer is a simple 2x rendering plugin that upscales the original video image to twice its size using nearest-neighbor filtering.

2xSaI

The "2xSaI" renderer is a 2x rendering plugin designed by Kreed that scales and interpolates the original image to provide a better-quality picture.

Blargg NTSC

Blargg's NTSC fitler is a 2x rendering plugin that simulates the image seen on a regular NTSC television set. Several different NTSC artifacting options are available, including color bleed and tint. A configuration window is available in the Plugins menu.
