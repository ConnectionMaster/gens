Gens/GS - Menus - File Menu

Index → Menus → Controller Configuration

The File Menu allows you to perform file operations.

Open ROM...: Open a ROM file or SegaCD disc image.

Boot CD: Boot a SegaCD game using a physical CD-ROM or CD-ROM drive emulator. This option is only available on Linux and Windows. Windows requires ASPI support. (On Windows 2000 and XP, you can use the Nero ASPI DLL; however, you must have administrative privileges in order to use ASPI.)

Netplay: Currently not available in Gens/GS.

ROM History: You can select a previously opened ROM image. ROM History stores up to 9 previously opened ROM images, which can also be loaded by using the key shortcut Ctrl+#, where # is the ROM number.

Close ROM: Closes the currently running ROM or SegaCD disc image.

Load State...: Allows you to select a savestate file to load.

Load State...: Allows you to create a savestate.

Quick Load: Loads a savestate from the current save slot for the loaded ROM image.

Quick Save: Saves a savestate to the current save slot for the loaded ROM image.

Change State: Allows you to change the current save slot number.

Quit: Closes Gens/GS.
